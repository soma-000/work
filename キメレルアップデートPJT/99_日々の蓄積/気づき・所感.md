# 業務の取り組み方
- 機能 + どう感じたのかを言語化して記載する
- 対象のページは先に開いておく
- 議事録のメモはシート上に記載する⇒野中さんにやってる感をアピールするため
- スマホでOOスクロールとかもデータとしてはありかも 
- 機能を第三者性を強化するか否かみたいな形で、カテゴライズするのもいいかも
# キメレルに関して
- 今キメレルは第三者が判定して、この企業がいいという押し出し方をしている。しかしユーザーからの口コミやレビューなど、ユーザー目線での声は取り入れられていない。
- また業界マッピングがないなど、情報の網羅性にも疑問が残る⇒初手で数社に絞られているのが違和感
- また最初に一社等の紹介など、押し出しが足りていないことも事実である
- クライアントのLPを遷移させずに、直接問い合わせに繋げる導線が作れれば熱い
# 野中さんとのMTGについて
- 時間がオーバーしがちなので、もう少し内容をコンパクトに説明できるといい
- 上から順に説明したい⇒スポットを先に設定して、それを基に説明
- メモもシートに残した方が後で見やすいため、自分用のメモと要点を両方記載し、見やすくシートを整理する
- やったことや相談したいことをテキストに起こすやり方はよかった⇒見やすそうにしてた
